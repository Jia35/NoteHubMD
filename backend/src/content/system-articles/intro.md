# 歡迎使用 NoteHubMD

NoteHubMD 是一個基於 Markdown 的協作筆記平台，讓您可以輕鬆地撰寫、分享和組織您的筆記。

## 主要功能

- **📝 Markdown 編輯器** - 即時預覽，所見即所得
- **📚 書本組織** - 將相關筆記整理成書本
- **🔗 分享功能** - 透過連結分享您的筆記
- **👥 協作編輯** - 多人即時編輯同一篇筆記
- **🎨 主題切換** - 支援深色/淺色模式

## 快速開始

1. 點擊左側邊欄的「+ 新增筆記」建立您的第一篇筆記
2. 使用 Markdown 語法撰寫內容
3. 在預覽區域查看渲染結果

> 💡 提示：按下 `Ctrl + S` 可以手動儲存筆記
